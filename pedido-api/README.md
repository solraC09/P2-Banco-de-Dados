# Pedido API

Projeto de API REST para gerenciamento de pedidos utilizando FastAPI, MongoDB, Docker, Kafka e RabbitMQ.

---

## Tecnologias utilizadas

- FastAPI
- MongoDB
- Docker & Docker Compose
- Kafka
- RabbitMQ
- Pytest
- Python 3.13

---

## Como executar o projeto

### 1. Subir os containers

docker compose up -d
